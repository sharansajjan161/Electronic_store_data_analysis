# 📊 Sales Data Analysis Project

A complete sales data analysis project built using Python, Pandas, and Matplotlib.
This project analyzes sales performance, customer ordering patterns, city-wise revenue, and product profitability using real-world style sales data.

---

## 🚀 Features

* ✅ Data Cleaning and Preprocessing
* ✅ Monthly Sales Analysis
* ✅ City-wise Revenue Analysis
* ✅ Peak Sales Time Detection
* ✅ Product Profitability Analysis
* ✅ Data Visualization using Matplotlib
* ✅ Exporting Graphs as High-Quality Images

---

## 🛠 Technologies Used

* Python
* Pandas
* Matplotlib
